# HTML to PDF Service

This is a small html-to-pdf service build over nodejs, express, phridge and phantomjs.

## Features:

* URL and HTML Payload requests.

## Start the service

``` chmod +x ./init.sh && ./init.sh ```

## Test the service

```  curl localhost:8080?url=http://google.com/ -o test.pdf ```
